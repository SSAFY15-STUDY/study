# 📝 실습 문제 02: 붙어 있는 숫자 분리하여 저장하기

입력된 한 줄의 숫자를 각각의 자릿수별로 나누어 정수형 배열(`int[]`)에 저장하는 프로그램을 구현합니다.

## 1. 문제 설명

사용자로부터 공백 없이 붙어 있는 숫자열(예: `12345`)을 입력받습니다. 이 숫자를 하나씩 분리하여 크기가 5인 정수형 배열에 순서대로 저장하고, 저장된 배열의 내용을 출력하여 확인하세요.

## 2. 입력 형식

1. 첫 번째 줄에 붙어 있는 숫자들이 입력됩니다. (최대 10자리 미만)

**입력 예시**

```text
12345

```

## 3. 출력 형식

배열에 저장된 각 요소를 공백으로 구분하여 출력하세요.

**출력 예시**

```text
1 2 3 4 5 

```
