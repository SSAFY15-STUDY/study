# 📝 실습 문제 02: 붙어 있는 숫자 분리하여 저장하기

입력된 한 줄의 숫자를 각각의 자릿수별로 나누어 정수형 배열(`int[]`)에 저장하는 프로그램을 구현합니다.

## 1. 문제 설명

사용자로부터 공백 없이 붙어 있는 숫자열(예: `12345`)을 입력받습니다. 이 숫자를 하나씩 분리하여 크기가 5인 정수형 배열에 순서대로 저장하고, 저장된 배열의 내용을 출력하여 확인하세요.

## 2. 입력 형식

1. 첫 번째 줄에 붙어 있는 숫자들이 입력됩니다. (최대 10자리 미만)

**입력 예시**

```text
12345

```

## 3. 출력 형식

배열에 저장된 각 요소를 공백으로 구분하여 출력하세요.

**출력 예시**

```text
1 2 3 4 5 

```

### 💡 힌트

1. **입력 받기**: `sc.next()`를 사용해 숫자열을 `String`으로 먼저 받으세요.
2. **분리 방법 (택 1)**:
* **방법 A**: `String.split("")`을 사용하여 한 글자씩 잘라 문자열 배열로 만든 뒤 `Integer.parseInt()`로 변환하기
* **방법 B**: `String.toCharArray()`로 `char[]`을 만든 뒤, 각 문자에 `'0'`을 빼서 정수로 변환하기 (예: `c - '0'`)


3. **배열 생성**: 입력받은 문자열의 길이(`length()`)만큼 배열 크기를 지정하세요.
